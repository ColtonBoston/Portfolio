Hello!

ListSharing

A web app for sharing collaborative lists online.

I'm really enjoying my time working on this application to make it easier and faster to get a grocery (or any other) list together. Anyone given permission by a list's owner can use their devices to add their own items to the list at their convenience.

This full-stack project has been an opportunity to learn and apply the technologies listed below, and I have learned so much about them during its development so far. The source code can be found on GitHub here.

All coding/designing on this app has been done by me, and I will continue with updates as I learn more and receive feedback. I hope you give it a try and find it useful!
